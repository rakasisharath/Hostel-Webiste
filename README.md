# Hostel Website

## Project Structure

- `index.html`: The main webpage for the hostel.
- `css/`: Contains stylesheet files for styling the website.
- `js/`: JavaScript files for interactivity.
- `images/`: Folder holding images used in the gallery.
- `pricing.json`: JSON file containing pricing details.
- `amenities.json`: JSON file listing all available amenities.

## Features

- **Room Gallery**: A section that showcases available rooms with images and descriptions.
- **Pricing Display**: Easy access to pricing information for different room types.
- **Amenities Information**: Details about amenities included with the stay, such as Wi-Fi, parking, breakfast, etc.

## Installation Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/rakasisharath/Hostel-Webiste.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Hostel-Webiste
   ```
3. Open `index.html` in a web browser to view the project.

## Usage Guidelines

- Navigate through the website using the navigation bar at the top.
- Click on the 'Room Gallery' link to explore available rooms.
- Visit the 'Pricing' section for detailed information on costs.
- Check the 'Amenities' section to understand what is included in the stay.
